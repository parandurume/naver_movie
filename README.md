- python3
- BeautifoulSoup4 (python3 module)

# Run code
- python3 crawl.py
- inside comments folder you will get the crawled comments
- comments폴더 안에 평점과 댓글이 저장됩니다.
    - comments 폴더가 꼭 있어야 합니다.
    - you must have commets folder.
